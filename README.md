# Inventory Management System

## Overview
The Inventory Management System is a simple web application designed to help store and manage products. It allows administrators to view, add, edit, and delete products. The system also supports basic user registration and login functionalities for admins.

## Features
- **Admin Login**: Admins can log in to access the dashboard.
- **Product Management**: Admins can add, update, and delete products.
- **Predefined Products**: The application comes with a set of predefined products for demonstration purposes.
- **Logout**: Admins can log out to securely end their session.
- **Responsive Design**: The application is responsive and works well on both desktop and mobile devices.

## Technologies Used
- **Frontend**: HTML, CSS, Bootstrap (for responsive design)
- **Backend**: PHP (for server-side processing)
- **Database**: MySQL (for storing user and product data)
- **Version Control**: Git & GitHub

## Installation Instructions

### Prerequisites
Make sure you have the following installed on your local machine:
- PHP (>= 7.0)
- MySQL (for database functionality)
- A local server like XAMPP, WAMP, or MAMP

### Steps to Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/niteshbassai2@gmail.com/inventory-management-system.git
2. Navigate to the project directory:

cd inventory-management-system


3. Set up the database:

Open phpMyAdmin (or your preferred MySQL tool).

Create a new database (e.g., inventory_db).

Run the SQL queries to create necessary tables for products and admin users.

4. Set up the project on your local server:

Place the project folder in the htdocs (for XAMPP) or www (for WAMP) folder.

Open localhost/your-project-folder in your web browser to access the application.

### Usage

To access the Admin Dashboard, log in with the credentials provided (admin panel functionality).

Logout is available in the top-right corner of the dashboard.

Add, edit, or delete products using the available buttons.
